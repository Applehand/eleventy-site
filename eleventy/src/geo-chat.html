AI Search Manual GEOAssistant

A multimodal RAG chatbot that answers strictly from the AI Search Manual — text & page-screenshot retrieval, streamed and grounded in citations.

..what?

This is a retrieval-augmented generation (RAG) demo. The manual's 23 chapters were re-crawled, heading-aware chunked, and embedded — both as text chunks and as full-page screenshots — with Google's gemini-embedding-2 into a Chroma vector database. Each question is embedded, the closest text/image chunks are retrieved, and gemini-3.5-flash-lite composes a streamed answer strictly from that retrieved context (it's instructed to say so if the context doesn't support an answer).

Compute: the vector store, embeddings, and generation all run on a self-hosted Raspberry Pi (bigpi, FastAPI + Chroma + Gemini API) on my home network. This page and a thin same-origin proxy live on a second Pi (smolpi), which is the only thing a Cloudflare Tunnel exposes publicly. Requests flow Browser → smolpi (public) → bigpi (Tailscale, private) and stream tokens back over SSE.

Free queries are capped per-session (cookie) and per-IP, enforced server-side — the controls below are real, but always clamped to safe ranges server-side regardless of what the client sends. Turn on Step-through to see retrieval happen before the model answers. Retrieved page screenshots render below each answer, and if the model can't answer from the retrieved context, a Why no answer? panel shows exactly what retrieval pulled and how close it got.
